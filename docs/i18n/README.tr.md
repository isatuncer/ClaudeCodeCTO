## Bu Nedir?

Kurulumdan sonra Claude Code, kurumsal duzeyde bir CTO'ya donusur. 1800+ beceri, 300+ ajan, 69 belge sablonu ve 72 kurumsal standart ile projenizi sifirdan planlar, gelistirir ve teslim eder. Tek yapmaniz gereken `/startCTO` yazmak.

## Hizli Baslangic

```bash
git clone --recursive https://github.com/YOUR_USER/ClaudeCodeCTO.git
cd ClaudeCodeCTO
bash setup.sh --all
```

Ardindan projenizde:
```
/startCTO
```

## Neler Yuklenir

| Kategori | Adet | Aciklama |

## Temel Komutlar

| Komut | Aciklama |
