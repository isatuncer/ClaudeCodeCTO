## 这是什么？

安装后，Claude Code 将成为企业级 CTO。拥有 1,800+ 技能、300+ 代理、69 个文档模板和 72 项企业标准，从零开始规划、开发和交付你的项目。只需输入 `/startCTO`。

## 快速开始

```bash
git clone --recursive https://github.com/YOUR_USER/ClaudeCodeCTO.git
cd ClaudeCodeCTO
bash setup.sh --all
```

然后在你的项目中：
```
/startCTO
```

## 安装内容

| 类别 | 数量 | 说明 |
|------|------|------|
| 技能 | 1,800+ | 覆盖所有语言和框架的编码模式 |
| 代理 | 300+ | 代码审查、安全、TDD、构建修复 |
| 模板 | 69 | 企业文档模板（SRS、HLD、ADR...） |
| 标准 | 72 | ISO、IEEE、OWASP、NIST、DORA |
| 图表 | 30 | Mermaid 图表模板（C4、ER、时序图...） |
| 角色 | 20 | 专家角色定义（从 CTO 到 DevSecOps） |

## 核心命令

| 命令 | 说明 |
|------|------|
| `/startCTO` | 启动自主 CTO 模式 |
| `/doc-create SRS` | 从模板生成文档 |
| `/doc-export` | 将 MD 导出为 PDF/DOCX |
| `/cto-status` | 系统状态面板 |
