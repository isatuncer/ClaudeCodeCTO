## Cos'e?

Dopo l'installazione, Claude Code diventa un CTO di livello enterprise. Con 1.800+ competenze, 300+ agenti, 69 modelli di documenti e 72 standard aziendali, pianifica, sviluppa e consegna il tuo progetto da zero. Basta digitare `/startCTO`.

## Avvio Rapido

```bash
git clone --recursive https://github.com/YOUR_USER/ClaudeCodeCTO.git
cd ClaudeCodeCTO
bash setup.sh --all
```

Poi nel tuo progetto:
```
/startCTO
```

## Cosa viene installato

| Categoria | Quantita | Descrizione |
|-----------|----------|-------------|
| Competenze | 1.800+ | Pattern di codice per ogni linguaggio e framework |
| Agenti | 300+ | Code review, sicurezza, TDD, fix build |
| Modelli | 69 | Documenti aziendali (SRS, HLD, ADR...) |
| Standard | 72 | ISO, IEEE, OWASP, NIST, DORA |
| Diagrammi | 30 | Template Mermaid (C4, ER, sequenza...) |
| Ruoli | 20 | Definizioni ruoli esperti (da CTO a DevSecOps) |

## Comandi Principali

| Comando | Descrizione |
|---------|-------------|
| `/startCTO` | Avvia modalita CTO autonoma |
| `/doc-create SRS` | Genera documento da template |
| `/doc-export` | Esporta MD in PDF/DOCX |
| `/cto-status` | Dashboard stato sistema |
