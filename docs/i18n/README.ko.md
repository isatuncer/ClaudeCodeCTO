## 이것은 무엇인가요?

설정 후 Claude Code는 엔터프라이즈급 CTO가 됩니다. 1,800개 이상의 스킬, 300개 이상의 에이전트, 69개의 문서 템플릿, 72개의 엔터프라이즈 표준으로 프로젝트를 처음부터 계획, 개발, 납품합니다. `/startCTO`만 입력하면 됩니다.

## 빠른 시작

```bash
git clone --recursive https://github.com/YOUR_USER/ClaudeCodeCTO.git
cd ClaudeCodeCTO
bash setup.sh --all
```

그런 다음 프로젝트에서:
```
/startCTO
```

## 설치되는 것들

| 카테고리 | 수량 | 설명 |
|---------|------|------|
| 스킬 | 1,800+ | 모든 언어와 프레임워크를 위한 코딩 패턴 |
| 에이전트 | 300+ | 코드 리뷰, 보안, TDD, 빌드 수정 |
| 템플릿 | 69 | 엔터프라이즈 문서 (SRS, HLD, ADR...) |
| 표준 | 72 | ISO, IEEE, OWASP, NIST, DORA |
| 다이어그램 | 30 | Mermaid 템플릿 (C4, ER, 시퀀스...) |
| 역할 | 20 | 전문가 역할 정의 (CTO에서 DevSecOps까지) |

## 주요 명령어

| 명령어 | 설명 |
|--------|------|
| `/startCTO` | 자율 CTO 모드 시작 |
| `/doc-create SRS` | 템플릿에서 문서 생성 |
| `/doc-export` | MD를 PDF/DOCX로 내보내기 |
| `/cto-status` | 시스템 대시보드 표시 |
