## Qu'est-ce que c'est ?

Apres l'installation, Claude Code devient un CTO de niveau entreprise. Avec 1 800+ competences, 300+ agents, 69 modeles de documents et 72 normes d'entreprise, il planifie, developpe et livre votre projet a partir de zero. Tapez simplement `/startCTO`.

## Demarrage Rapide

```bash
git clone --recursive https://github.com/YOUR_USER/ClaudeCodeCTO.git
cd ClaudeCodeCTO
bash setup.sh --all
```

Puis dans votre projet :
```
/startCTO
```

## Ce qui est installe

| Categorie | Nombre | Description |
|-----------|--------|-------------|
| Competences | 1 800+ | Modeles de code pour chaque langage et framework |
| Agents | 300+ | Revue de code, securite, TDD, correction de builds |
| Modeles | 69 | Documents d'entreprise (SRS, HLD, ADR...) |
| Normes | 72 | ISO, IEEE, OWASP, NIST, DORA |
| Diagrammes | 30 | Modeles Mermaid (C4, ER, sequence...) |
| Roles | 20 | Definitions de roles experts (CTO a DevSecOps) |

## Commandes Principales

| Commande | Description |
|----------|-------------|
| `/startCTO` | Lancer le mode CTO autonome |
| `/doc-create SRS` | Generer un document a partir d'un modele |
| `/doc-export` | Exporter MD en PDF/DOCX |
| `/cto-status` | Tableau de bord systeme |
