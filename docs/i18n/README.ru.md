## Что это?

После установки Claude Code становится CTO корпоративного уровня. С 1 800+ навыками, 300+ агентами, 69 шаблонами документов и 72 корпоративными стандартами он планирует, разрабатывает и поставляет ваш проект с нуля. Просто введите `/startCTO`.

## Быстрый старт

```bash
git clone --recursive https://github.com/YOUR_USER/ClaudeCodeCTO.git
cd ClaudeCodeCTO
bash setup.sh --all
```

Затем в вашем проекте:
```
/startCTO
```

## Что устанавливается

| Категория | Количество | Описание |
|-----------|-----------|----------|
| Навыки | 1 800+ | Шаблоны кода для каждого языка и фреймворка |
| Агенты | 300+ | Ревью кода, безопасность, TDD, исправление сборки |
| Шаблоны | 69 | Корпоративные документы (SRS, HLD, ADR...) |
| Стандарты | 72 | ISO, IEEE, OWASP, NIST, DORA |
| Диаграммы | 30 | Шаблоны Mermaid (C4, ER, последовательности...) |
| Роли | 20 | Определения экспертных ролей (от CTO до DevSecOps) |

## Основные команды

| Команда | Описание |
|---------|----------|
| `/startCTO` | Запустить автономный режим CTO |
| `/doc-create SRS` | Создать документ из шаблона |
| `/doc-export` | Экспортировать MD в PDF/DOCX |
| `/cto-status` | Панель состояния системы |
