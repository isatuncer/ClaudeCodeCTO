## これは何ですか？

セットアップ後、Claude CodeはエンタープライズグレードのCTOになります。1,800以上のスキル、300以上のエージェント、69のドキュメントテンプレート、72のエンタープライズ標準を備え、プロジェクトをゼロから計画・開発・納品します。`/startCTO`と入力するだけです。

## クイックスタート

```bash
git clone --recursive https://github.com/YOUR_USER/ClaudeCodeCTO.git
cd ClaudeCodeCTO
bash setup.sh --all
```

その後、プロジェクトディレクトリで：
```
/startCTO
```

## インストールされるもの

| カテゴリ | 数量 | 説明 |
|---------|------|------|
| スキル | 1,800+ | すべての言語・フレームワーク向けコーディングパターン |
| エージェント | 300+ | コードレビュー、セキュリティ、TDD、ビルド修正 |
| テンプレート | 69 | エンタープライズ文書（SRS、HLD、ADR...） |
| 標準規格 | 72 | ISO、IEEE、OWASP、NIST、DORA |
| ダイアグラム | 30 | Mermaidテンプレート（C4、ER、シーケンス...） |
| ロール | 20 | 専門家ロール定義（CTOからDevSecOpsまで） |

## 主要コマンド

| コマンド | 説明 |
|---------|------|
| `/startCTO` | 自律CTOモードを起動 |
| `/doc-create SRS` | テンプレートからドキュメントを生成 |
| `/doc-export` | MDをPDF/DOCXにエクスポート |
| `/cto-status` | システムダッシュボードを表示 |
