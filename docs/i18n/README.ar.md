## ما هذا؟

بعد التثبيت، يصبح Claude Code مديرًا تقنيًا بمستوى المؤسسات. مع أكثر من 1,800 مهارة، و300+ وكيل، و69 قالب مستندات، و72 معيارًا مؤسسيًا، يخطط مشروعك ويطوره ويسلمه من الصفر. فقط اكتب `/startCTO`.

## البدء السريع

</div>

```bash
git clone --recursive https://github.com/YOUR_USER/ClaudeCodeCTO.git
cd ClaudeCodeCTO
bash setup.sh --all
```

<div dir="rtl">

ثم في مشروعك:

</div>

```
/startCTO
```

<div dir="rtl">

## ماذا يتم تثبيته

| الفئة | العدد | الوصف |
|-------|-------|-------|
| المهارات | +1,800 | أنماط برمجة لكل لغة وإطار عمل |
| الوكلاء | +300 | مراجعة الكود، الأمان، TDD، إصلاح البناء |
| القوالب | 69 | مستندات مؤسسية (SRS، HLD، ADR...) |
| المعايير | 72 | ISO، IEEE، OWASP، NIST، DORA |
| المخططات | 30 | قوالب Mermaid (C4، ER، تسلسلي...) |
| الأدوار | 20 | تعريفات أدوار الخبراء (من CTO إلى DevSecOps) |

## الأوامر الرئيسية

| الأمر | الوصف |
|-------|-------|
| `/startCTO` | تشغيل وضع CTO المستقل |
| `/doc-create SRS` | إنشاء مستند من قالب |
| `/doc-export` | تصدير MD إلى PDF/DOCX |
| `/cto-status` | لوحة حالة النظام |
