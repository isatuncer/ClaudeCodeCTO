## Was ist das?

Nach der Einrichtung wird Claude Code zu einem unternehmenstauglichen CTO. Mit 1.800+ Skills, 300+ Agenten, 69 Dokumentvorlagen und 72 Unternehmensstandards plant, entwickelt und liefert er Ihr Projekt von Grund auf. Geben Sie einfach `/startCTO` ein.

## Schnellstart

```bash
git clone --recursive https://github.com/YOUR_USER/ClaudeCodeCTO.git
cd ClaudeCodeCTO
bash setup.sh --all
```

Dann in Ihrem Projekt:
```
/startCTO
```

## Was wird installiert

| Kategorie | Anzahl | Beschreibung |
|-----------|--------|-------------|
| Skills | 1.800+ | Coding-Muster fuer jede Sprache und Framework |
| Agenten | 300+ | Code-Review, Sicherheit, TDD, Build-Fix |
| Vorlagen | 69 | Unternehmensdokumente (SRS, HLD, ADR...) |
| Standards | 72 | ISO, IEEE, OWASP, NIST, DORA |
| Diagramme | 30 | Mermaid-Vorlagen (C4, ER, Sequenz...) |
| Rollen | 20 | Expertenrollen (CTO bis DevSecOps) |

## Wichtige Befehle

| Befehl | Beschreibung |
|--------|-------------|
| `/startCTO` | Autonomen CTO-Modus starten |
| `/doc-create SRS` | Dokument aus Vorlage erstellen |
| `/doc-export` | MD nach PDF/DOCX exportieren |
| `/cto-status` | System-Dashboard anzeigen |
