## How to regenerate

```bash
# 1. Rescan sources/ submodules into decisions/catalog.json
python scripts/extractor.py

# 2. Generate a new dated snapshot in this directory
python scripts/generate_manuals.py
```

Use `--date=DD-MM-YYYY` to override the timestamp, or `--force` to overwrite an existing snapshot.
